Atributos

  • 'value' es para darle un valor de inicio determinado
  • 'readonly' sirve para que sea solo lectura
  • 'disable' desabilita completamente el campo
  • 'require' marca que ese campo es requerido
  • 'autofocus' el puntero inicia en ese campo
